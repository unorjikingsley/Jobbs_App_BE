# 📖 JOB App <a name="about-project"></a>

**JOB App:** An application crafted with the MERN (MongoDB, Express.js, React, Node.js) stack designed to streamline the meticulous organization of job applications while providing a comprehensive framework for tracking individual progress within each professional pursuit.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- `React`
- `Redux`
- `Node`
- `Code editor (VSCode, RubyMine)`

### Setup

Clone this repository to your desired folder:

```sh
  git clone https://github.com/unorjikingsley/Jobbs_App_BE.git
```

### Install

Install this project run these commands in order:

```sh
  cd my-project
  npm install
  npm run dev
```

Also create an .env file with the following information

```sh
  NODE_ENV 
  PORT 
  MONGO_URL 
  JWT_SECRET 
  JWT_EXPIRES_IN

  CLOUD_NAME 
  CLOUD_API_KEY 
  CLOUD_API_SECRET
```
